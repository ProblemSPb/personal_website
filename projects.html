Projects

Here are some examples of my projects. Most of them are pet projects as work projects cannot be revealed due to the sensitivity of the human resources and employees related data.

Employment Contracts Generator

This is an automation project for employment contract generation. In order to release the burden from HR Operations Team I have written the JavaScript code integrated into a google sheet file. When the respective Google Sheet document is opened, it pulls a bound employment contract template to auto generate contracts for each new starter in the google sheet table.

Tech: JavaScript, Google Apps Script

T-Shirts Shop

T-Shirt Shop is a template website for shops that sell t-shirts with various prints. A user can choose an item and play the colors of the t-shirt. I have done this pet project to master HTML and CSS skills. Please, note it's not a production web site.

Go to project's website

Tech: HTML, CSS, JavaScript

Tokio Booking

The project is about finding a house or a flat for rent. It shows available options when a user moves a pin on the map. By clicking on the rendered ad pins users can see detailed information about every offer, uncluding pictures of the place. Users can also filter offers based on their preferences (price, number of rooms, etc) It was a pet project to learn JavaScript.

Tech: JavaScriptL

People Analytics Dashboards in Tableau

I have built four sets of interactive Tableau dashboards that connect recruiting and attrition data, collected from PeopleSoft and the company's ATS tool. Interactive dashboards help the organisation's leaders uncover hidden insights and lead to data driven business decisions.

Tech: Tableau Desktop, MySQL

Yeti Cave

Yeti Cave is an auction platform for selling and buying used snowboarding equipment. Users can place bids and see the hostory of the bods for a selected item. After the end of the auction, the service sends an auto notification to the winner's email with the details.

Tech: PHP, MySQL

Tasks Manager

This is a web service to manage your TODO lists and projects. Create a profile and schedule, assign deadlines to your upcoming tasks, upload documents to the tasks. It was a pet project to practice my PHP and MySQL skills.

Tech: PHP, MySQL
